SDGolf Monitor — SMS opt-in workflow

Last updated July 14, 2026

This page publicly documents the complete SMS opt-in workflow for SDGolf Monitor text alerts. The application itself is a private, invite-only service for a small group of friends and requires a personal login, so the enrollment form cannot be reached anonymously. This page and the privacy policy are published without any login specifically so that carrier and messaging-compliance reviewers can verify the opt-in process end to end.

About the program

  • Program name: SDGolf Monitor.
  • What it sends: purely informational alerts — a text when a new San Diego golf tee time matches a search the user set up, with a booking link. No marketing or promotional content of any kind.
  • Audience: a small, invite-only group of friends (accounts are allow-listed by the maintainer). Expected volume is under 500 messages per month.
  • Scope: US phone numbers only. Message frequency varies with tee-time availability. Message and data rates may apply.

The opt-in workflow, step by step

  1. Sign in. The user signs in to their invite-only account at the application (the site root shows this login — it is the password-protected part of the service).
  2. Open “Texts”. The user clicks the Texts button in the app’s navigation, which opens the enrollment form shown in the screenshot below. The user types their own mobile number. The consent checkbox is unchecked by default, and the submit button stays disabled until the user checks it. The checkbox carries this exact disclosure:
    Text me new tee-time alerts from SDGolf Monitor. Message frequency varies. Msg & data rates may apply. Reply STOP to cancel, HELP for help. See the privacy policy.
  3. Verification text (double opt-in). Submitting the form sends a single verification message to the number the user entered:
    SDGolf Monitor: your code is 482913. Alerts for new tee times; freq varies. Msg&data rates may apply. Reply STOP to opt out, HELP for help.
  4. Code entry. The user types the 6-digit code back into the app to prove possession of the phone. Only after this verification succeeds is the number eligible to receive alerts — an unverified number is never texted again.

Sample alert message

After verification, the only recurring message type is the new-tee-time alert:

SDGolf: Torrey Pines North Sat 7/18 7:20 AM (+2 more) Txt STOP to end https://foreupsoftware.com/index.php/booking/19347#/teetimes

Opt-out and help

  • Reply STOP to any message to cancel immediately (handled at the carrier level; texting START later re-enables delivery, after which the user must re-verify in the app).
  • Turn text alerts off any time in the app via the same “Texts” dialog.
  • Reply HELP to any message, or email sdgolfmonitor@gmail.com.
  • Email alerts continue independently either way — SMS is an optional add-on.

Consent records

Each opt-in stores a consent record — timestamp, IP address, browser user-agent, and the exact version of the disclosure text agreed to — retained for at least 5 years, as described in the privacy policy. Phone numbers are never shared with third parties or affiliates for marketing or promotional purposes.

Questions, help, or data requests: sdgolfmonitor@gmail.com
